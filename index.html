Der Workshop

Der Workshop wird sich um MongoDB drehen (welch ein Wunder). Dieser wird einen theoretischen Teil haben (Vortrag von mir) + einen praktischen Teil (auf die Tastatur tippern von euch). Damit der letzte Teil problemlos vonstatten gehen kann, würde ich euch bitten euren Rechner diesbezüglich vorzubereiten. Jeder von euch muss MongoDB installieren. Da wir im praktischen Teil nicht nur direkt mit der Datenbank reden werden, sondern uns auch angucken wollen, wie das aus einer Anwendung aussehen würde, muss jeder Java + Maven oder Node.JS installieren. Die Anwendungsübungen sind also in Java oder in JavaScript zu lösen (was euch lieber gefällt, wobei ich die JavaScript Variante bevorzügen würde). Im Zweifel installiert Java, Maven und Node.JS (und da die meisten eh schon Java+Maven installiert haben, packt euch einfach Node.JS noch drauf)

MongoDB installieren

Installation auf dem Mac

Auf dem Mac kann Homebrew für die Installation von MongoDB genutzt werden. 

$ brew install mongo

Fertige Installations-Pakete für Windows/Linux/Mac werden direkt von der offiziellen MongoDB Seite zur Verfügung gestellt. Link zu MongoDB
