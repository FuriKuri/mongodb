Aufgabe 8

Ladet euch die Datei small_zips.json runter.
mongoimport -d test -c zips --drop small_zips.json

Berechnet die durchschnittliche Einwohneranzahl für die Städte in den Staaten California (CA) und New York (NY). In die Berechnung sollen nur Städte einfließen mit einer Einwohneranzahl größer als 25.000.
Achtung:
  • Städte können mehrere ZipCodes haben
  • Städte können in mehreren Staaten auftauchen
Für Städte, welche in mehr als nur einem Staat auftauchen gilt, werden als eigenständige Städte angesehen.
Hinweis: Für CT und NJ ist die Anwort: 38177

Aufgabe 11

Baut eine kleine Replica Set auf. 
mkdir -p /data/rs1 /data/rs2 /data/rs3

mongod --replSet m101 --logpath "1.log" --dbpath /data/rs1 --port 27017 --smallfiles --oplogSize 64 --fork
mongod --replSet m101 --logpath "2.log" --dbpath /data/rs2 --port 27018 --smallfiles --oplogSize 64 --fork
mongod --replSet m101 --logpath "3.log" --dbpath /data/rs3 --port 27019 --smallfiles --oplogSize 64 --fork
Windows Users: kein -p bei mkdir. Auch kein --fork. Zudem beim Pfad aufpassen, Bacl Slashes verwenden (e.g; C:\data\rs1). 
mongo --port 27017

config = { _id: "m101", members:[
          { _id : 0, host : "localhost:27017"},
          { _id : 1, host : "localhost:27018"},
          { _id : 2, host : "localhost:27019"} ]
         };
rs.initiate(config);
rs.status()
